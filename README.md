# 图片配乐视频生成器

选择一段背景音乐和多张图片，按照列表顺序生成 MP4 视频。默认每隔 4 秒切换到下一张图片，支持淡入淡出、滑动、圆形展开、溶解等转场。

## 环境准备

1. 安装 Python 3.10 或更高版本。
2. 安装 FFmpeg，并确保命令行可运行 `ffmpeg -version`。

Windows 可以执行：

```powershell
winget install Gyan.FFmpeg
```

安装后重新打开终端。也可以把 `ffmpeg.exe` 放到项目的 `tools` 文件夹中。

## 启动

双击项目目录中的 `启动视频工具.vbs`，或双击桌面上的“图片配乐视频生成器”快捷方式。启动时不会出现命令行窗口。

在界面中：

1. 添加图片；单击缩略图选中，双击放大查看，使用“上移 / 下移”调整播放顺序。
2. 选择背景音乐和转场方式。
3. 选中每张图片并设置其上方、下方时间文字。新图片默认分别为“十年前”和“现在”，文字会显示在画面中线两侧。
4. 选择输出路径，点击“生成视频”；点击“打开目录”可直接进入输出文件夹。

背景音乐短于视频时会自动循环，长于视频时会自动截断。所有图片都会等比例缩放，空白区域使用黑色填充。
